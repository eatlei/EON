# Subscribe

一个 SwiftUI iOS 订阅管理 App 原型，覆盖订阅记录、精美总览、清晰扣费日历、统一币种查看、iCloud 同步和续费提醒。

## 功能

- 管理常用订阅：名称、套餐、分类、状态、支付方式、席位、备注。
- 精美 UI：定制视觉系统、独立右下角新增按钮、数字过渡和轻量入场动效。
- 查看订阅消费：月/年一级切换、周期总额、扣费笔数、下一笔订阅、本月扣费日历、全年扣费分布。
- 清晰数据面板：支出分类饼图、本月收据式明细、近期明细。
- 订阅排序：支持按续费时间、周期时长、费用和名称查看订阅。
- 多币种统一统计：支持 CNY、USD、EUR、JPY、GBP、HKD，并以内置汇率换算。
- 复杂订阅周期：每周、每月、每季度、每年和自定义天数。
- 续费提醒：为订阅设置提前提醒天数，通过 iOS 本地通知提醒。
- iCloud 同步：通过 iCloud Key-Value Store 同步订阅数据。

## 运行

```bash
xcodegen generate
open Subscribe.xcodeproj
```

在 Xcode 中选择 `SubscribeApp` scheme，运行到 iOS 17+ 模拟器或真机。
